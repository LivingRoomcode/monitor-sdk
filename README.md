# Monitor SDK

前端监控 SDK

## 安装
<font color="red"> 已经发布，下面这行命令可以安装 </font>

```npm i @ni0duann/monitor-sdk ```

## 环境测试
```
  pnpm run build
  node test/test.js
```

## 使用方法
```typescript jacscript
import Monitor from 'monitor-sdk';

const monitor = new Monitor({
  // 配置项
});
```

## 功能特性
- 错误监控
- 性能监控
- 用户行为监控
